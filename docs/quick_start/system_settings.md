## 系统设置

!!! info ""
    * 仓库协议: 支持 http 和 https，默认 http
    * 仓库 IP: 默认为部署 KubeOperator 的服务器 IP。将使用该 IP 来访问 nexus 仓库
    * NTP Server: 时间同步服务器，默认可以为空。也可以自建或使用公共 NTP Server
